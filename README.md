# Development

## Installation
1. [Install Jekyll](http://jekyllrb.com/docs/installation/)
2. [Install NPM](https://nodejs.org/en/download/)

## Running Locally
1. `npm install`
2. `bower install`
3. `jekyll serve`
4. `gulp`

I would suggest running 3 and 4 at the same time. Gulp listens for changes and jekyll will serve them up.
